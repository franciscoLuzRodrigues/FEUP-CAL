Instruções de compilação:

O projeto foi desenvolvido no Eclipse, por esse motivo deverá ser compilado no mesmo. 
Para isso é necessário criar um projeto, cujo compilador é o MinGW, e copiar todo o código para a pasta src e ficheiros para a própria pasta criada pelo Eclipse.
A hierarquia da pasta do projeto submetido terá que se manter.
Após aberto o projeto terá de ser adicionada a biblioteca wsock32 de forma a este compilar.
Esta biblioteca é adicionada em Project -> Properties -> C/C++ build -> Settings -> MinGW C++ Linker -> Libraries.
Após estes passos está pronto a correr!

Ficheiros:
Para o projeto usámos os ficheiros disponibilizádos pelos professores. Estes encontram-se na pasta T05/ ao mesmo nivel de src/,
 a alteração da localizaçaão dos mesmos vai inviabilizar a sua leitura, consequentemente a execução dos algoritmos do programa.
